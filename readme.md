# CS2Gen

CS2 copy `!gen` creates a button on different websites (buff.163.com, csfloat.com, bitskins.com, steam.com/*/inventory). This button allows you to directly copy the `!gen` command to inspect skins on a Counter-Strike 2 server.

## Extension

[Extension link](https://chromewebstore.google.com/detail/cs2-copy-gen/chhjlbeglapjmmfekjoglbfpmbogcgce?hl=es&authuser=0)

## FAQ

### What do we use to get the `!gen` command?

We use the [CS2Inspects API](https://cs2inspects.com/).
